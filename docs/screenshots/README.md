# Screenshots

Emplacement des captures d'écran référencées par le `README.md` racine.

Déposer ici les fichiers suivants (mêmes noms → le README les affichera automatiquement) :

| Fichier             | Écran attendu                                             |
| ------------------- | --------------------------------------------------------- |
| `journal.png`       | Onglet 📅 Journal (totaux du jour + mini-calendrier)       |
| `coach.png`         | Onglet 💬 Coach (chat, modes 📝 Info / 💬 Question)         |
| `coach-chart.png`   | Un graphique Recharts généré inline dans le chat          |
| `photos.png`        | Onglet 📸 Photos (comparateur avant/après à curseur)       |
| `exercises.png`     | Onglet 🏋️ Exos (liste filtrée + images)                    |

Format conseillé : PNG, largeur ~1080 px (rendu mobile PWA), fond réel de l'app.
Aucune donnée sensible sur les captures (email de connexion, etc.).
